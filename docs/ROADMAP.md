# Vulcan Roadmap

## M1

- SSE -> WebSocket 전환
- OpenClaw 신호 수집 고도화 (공식 API/CLI 우선)
- Office 상태 애니메이션 정교화
- 이벤트 타입 확장(message/tool_call/error 외)

## M2

- proactive memory 실험(mem0 / supermemory / memU 비교)
- placeholder를 자체 제작 pixel asset으로 교체
- 팀 운영 지표(throughput, retry, queue pressure) 시각화

## Out of Scope (M0 유지)

- 무거운 승인/감사 체계
- 정책 엔진
- 복잡한 RBAC
